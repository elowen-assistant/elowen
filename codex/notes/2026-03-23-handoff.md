## Important Product Notes

- Posting a message to a thread currently persists the message only.
- Elowen does not auto-reply in-thread yet.
- The active runtime interaction model is still job-driven:
  - post message to thread
  - create job from thread
  - inspect execution, summary, approvals, and notes
